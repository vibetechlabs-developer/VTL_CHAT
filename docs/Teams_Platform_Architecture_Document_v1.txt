Teams-Like Internal Collaboration Platform
Architecture Document

System Overview
Frontend: React + Vite
Backend: Django + DRF
Realtime: Django Channels + Redis
Database: PostgreSQL
Calls/Meetings: WebRTC + STUN/TURN
Storage: Object Storage
Deployment: Docker + Nginx
